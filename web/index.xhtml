ScoreTracker

Welkom bij Team 1

Dit is ons project score tracker voor JAVA JEE.

Deze webapplicatie wordt gebruikt voor de punten van student in te geven. De punten worden door de leerkracht ingegeven, door middel van een excel bestand.

De applicatie zal deze punten verwerken en geeft de mogelijkhijd om een rapport te genereren in pdf formaat.
